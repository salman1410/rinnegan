### RINNEGAN
information gathering tool for pentesters

### REQUIREMENTS
* `python`(Version 2 or 3)
* `pip`(Python Package Manager)

### SETUP & USAGE
* `git clone https://github.com/salman1410/rinnegan/`
* `cd rinnegan`
* `python rinnegan.py`

### FEATURES
* Whois Lookup
* DNS Lookup
* Cloudflare Detector
* HTTP Header Grabber

### NOTE
* Tested on KALI LINUX 2017 & WINDOWS 10
